# Raspberrypi-3b-mini-project
### Raspberrypi 3b+ mini project using python in jan, 2020 at KNU IT edu_center

2020년 1월 경북대학교 IT 교육센터에서 진행한 라즈베리파이3 미니 프로젝트 수업 수강 내용입니다.

사용언어 : Python

사용한 라즈베리파이 버전 : raspberryPi 3b



# 일자별 수업 내용
